|------------------------------------ Opitional ----------------------------------------|
|	1-) Multilanguage System.						     	|
|------------------------------------- How to use --------------------------------------|
|	1-) Put the M_ShopManager on some object of the Scene (preference on an object 	|	
| that will not destroy on load.)						     	|
|	1-) Put the M_ItemDataManageron some object of the Scene(preference on an object|	
| that will not destroy on load.)						     	|
|	3-) Create the ItemDatabase asset and populate them. You will need the following|
| things. 										|
|				!!! IMPORTANT!!						|
|	  (Right click/Create/Data/Mup/ShopSystem/Create GenericItem Database)  	|
|				!!! IMPORTANT!!						|
|											|
|	  - Add the currences that you will use on the game				|
|	  - Create all items and assign the values 				  	|
|	  REMEMBER DO NOT USE SPACE IN THE ITEM NAME. USE _ 			  	|
|	4-) If you'll use the Multilanguage system remember:				|
|	  - Create 1 field for each item name with the following example:		|
|		@shop_system_item_name_PRODUCTID					|
|	  - Create 1 field for each item description with the following example:	|
|		@shop_system_item_desc_PRODUCTID					|
